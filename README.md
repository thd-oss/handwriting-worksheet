# ✏️ 바른 글씨 쓰기 학습지 만들기

초등학생 맞춤형 **원고지 / 줄공책 학습지**를 브라우저에서 만들고 바로 인쇄할 수 있는 웹 앱(PWA)입니다.  
설치 없이 **HTML 파일 하나**만 다운로드하면 바로 사용할 수 있습니다.

---

## ⬇️ 다운로드

> 아래 링크를 클릭하면 최신 버전이 다운로드됩니다.

- **[📥 handwriting-worksheet.html 직접 다운로드](https://github.com/thd-oss/handwriting-worksheet/raw/main/handwriting-worksheet.html)**
- **[📦 최신 릴리즈 페이지 (v1.1.0)](https://github.com/thd-oss/handwriting-worksheet/releases/latest)**

---

## 🚀 사용 방법

1. 위 링크에서 `handwriting-worksheet.html` 파일을 다운로드합니다.
2. 다운로드된 파일을 **더블클릭**하여 브라우저에서 엽니다.
3. 왼쪽 설정창에서 학교명/학년/이름 및 텍스트와 서식을 입력합니다.
4. 우측 미리보기를 확인한 후 **인쇄 버튼**을 누릅니다.

> ⚠️ **인터넷 연결이 필요합니다.** 폰트를 Google Fonts에서 불러오기 때문입니다.  
> 오프라인 환경에서 사용하려면 파일 내 주석의 안내를 따라 폰트를 로컬에 저장하세요.

## ✨ 주요 기능

| 기능 | 설명 |
|---|---|
| **PWA & 자동 업데이트** | 서비스 워커 기반 오프라인 지원 및 새 배포 시 자동 알림 |
| **원고지 양식** | 칸 수, 글자 크기, 폰트 종류, **보조 십자선 ON/OFF** |
| **A4 페이지 자동 분리** | 긴 문장 작성 시 A4 규격에 맞춰 2페이지 이상 수직 분리 |
| **줄공책 양식** | 일반 텍스트 입력, 글자 크기와 연동된 줄 간격 |
| **번갈아 빈 줄 삽입** | 각 줄 아래 연습용 빈 줄을 1~3줄 추가 |
| **헤더 설정 통합** | 학교명, 학년, 반, 이름, 주제를 설정창에서 일관되게 편집 |
| **맞춤법 검사** | 클립보드 복사 후 네이버/다음/인크루트 검사기 선택 연동 |
| **인쇄 최적화** | A4 용지 기준 여백/레이아웃 및 페이징 자동 처리 |

# ✏️ 바른 글씨 쓰기 학습지 만들기

초등학생 맞춤형 **원고지 / 줄공책 학습지**를 브라우저에서 만들고 바로 인쇄할 수 있는 웹 앱입니다.  
설치 없이 **HTML 파일 하나**만 다운로드하면 바로 사용할 수 있습니다.

---

## ⬇️ 다운로드

> 아래 버튼을 클릭하면 파일이 바로 다운로드됩니다.

**[📥 handwriting-worksheet.html 다운로드](https://github.com/thd-oss/handwriting-worksheet/raw/main/handwriting-worksheet.html)**

또는 [릴리즈 페이지](https://github.com/thd-oss/handwriting-worksheet/releases/latest)에서도 다운로드할 수 있습니다.

---

## 🚀 사용 방법

1. 위 링크에서 `handwriting-worksheet.html` 파일을 다운로드합니다.
2. 다운로드된 파일을 **더블클릭**하여 브라우저에서 엽니다.
3. 왼쪽 패널에서 원하는 설정을 조작합니다.
4. 우측 미리보기를 확인한 후 **인쇄 버튼**을 누릅니다.

> ⚠️ **인터넷 연결이 필요합니다.** 폰트를 Google Fonts에서 불러오기 때문입니다.  
> 오프라인 환경에서 사용하려면 파일 내 주석의 안내를 따라 폰트를 로컬에 저장하세요.

---

## ✨ 주요 기능

| 기능 | 설명 |
|---|---|
| **원고지 양식** | 칸 수, 글자 크기, 폰트 종류 자유 설정 |
| **줄공책 양식** | 일반 텍스트 입력, 글자 크기와 연동된 줄 간격 |
| **번갈아 빈 줄 삽입** | 각 줄 아래 연습용 빈 줄을 1~3줄 추가 |
| **다양한 폰트** | 나눔고딕, 나눔명조, 나눔손글씨 등 지원 |
| **맞춤법 검사** | 텍스트를 클립보드에 복사 후 외부 검사기 연동 |
| **인쇄 최적화** | A4 용지 기준 여백/레이아웃 자동 설정 |
